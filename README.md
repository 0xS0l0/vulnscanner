# 🛡️ Web Vulnerability Scanner

A lightweight **Python-based web vulnerability scanner** with an interactive **Flask web interface** designed to automate the detection of common web application vulnerabilities. The application performs automated security assessments for **SQL Injection (SQLi)** and **Cross-Site Scripting (XSS)** using custom payloads and response analysis, presenting the results through a simple web dashboard.

## 🚀 Features

- 🌐 User-friendly Flask web interface
- 🔍 Automated SQL Injection (SQLi) detection
- 🛡️ Cross-Site Scripting (XSS) detection
- ⚙️ Custom payload execution
- 📡 HTTP request and response analysis
- 📄 Structured vulnerability reporting
- 🐍 Lightweight Python implementation
- 🔌 Modular architecture for adding new vulnerability checks

## 🏗️ Built With

- Python
- Flask
- Requests
- BeautifulSoup4
- HTML Parsing
- Regular Expressions

## 🎯 Purpose

This project was developed to improve practical knowledge of **web application security**, **offensive security automation**, and the **OWASP Top 10**. It demonstrates how automated tooling can assist penetration testers by reducing repetitive testing tasks while providing an intuitive web interface for managing scans and reviewing results.
